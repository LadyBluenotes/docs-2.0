# Track My Funds

<Hero src="/TMF.png" alt="Screen shot of track my funds website home page." repo="https://github.com/LadyBluenotes/track-my-funds" href="https://www.trackmyfunds.app/" />


Track My Funds is a full stack personal finance application that uses Next.js, TypeScript, MongoDB, Tailwind CSS, and Next-Auth. It offers a sleek interface and valuable insights to empower users in managing their finances effectively.

## I. Getting started

### Prerequisites

In addition to an IDE you must have the following:

- [Node.js](https://nodejs.org/en)
- [MongoDB Account](https://www.mongodb.com/)
- [Google Account for Next-Auth](https://next-auth.js.org/providers/google)

### Installation

How to install and run the application.

1. Clone the repository from GitHub.
```bash copy
        git clone https://github.com/LadyBluenotes/track-my-funds
```

2. Install Node packages
```bash copy
        npm install
```

3. Copy the `.env.example` file and input the environment variables.

4. Run the application
```bash copy
        npm run dev
```
